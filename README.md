<h1>크리스마스 이벤트 웹페이지</h1>

<p><strong>서버 주소:</strong> https://christmastree-dccfb.web.app/index.html</p>

<br>

<p>
&lt;소개&gt;<br>
저희가 구현한 웹페이지는 '크리스마스 이벤트 웹페이지'입니다.<br>
해당 웹페이지는 다가오는 크리스미스를 기대하며, 다양한 이벤트를 즐길 수 있도록 만들었습니다.<br>
저희가 준비한 주요 기능은 '편지 보내기', '트리 꾸미기', '미니게임'이 있으며, 그 외에도 '눈내리기', '음악 듣기' 등의 기능이 있습니다.<br>
자세한 기능은 아래에서 소개하겠습니다.
</p>

<br>

<p>
&lt;기능&gt;<br><br>

1. 전체 페이지<br>
&nbsp;&nbsp;&nbsp;&nbsp;페이지 전체에서 사용할 수 있는 기능은 '눈내리기'와 '음악 듣기'입니다.<br>
&nbsp;&nbsp;&nbsp;&nbsp;모든 페이지 하단 오른쪽에는 각 기능을 수행할 수 있는 동그란 버튼이 있습니다.<br>
&nbsp;&nbsp;&nbsp;&nbsp;🔈 모양의 버튼을 누르면 크리스마스 캐롤이 재생되며, 해당 버튼(🔊)을 다시 누르면 음소거가 됩니다.<br>
&nbsp;&nbsp;&nbsp;&nbsp;☁️ 모양의 버튼을 누르면 페이지 전체에 눈이 내리며, 해당 버튼(❄️)을 다시 누르면 눈이 사라집니다.<br>
&nbsp;&nbsp;&nbsp;&nbsp;사용자가 원하는 대로 버튼을 조작하며, 페이지를 풍부하게 즐길 수 있습니다.<br>
&nbsp;&nbsp;&nbsp;&nbsp;상단에는 네비게이션바가 있어, 원하는 페이지로 바로 이동할 수 있게 하였습니다.<br><br>

3. 메인 페이지<br>
&nbsp;&nbsp;&nbsp;&nbsp;메인 페이지에는 크리스마스와 어울리는 이미지 카드와 그 안에 'Merry chritmas' 문구를 넣어 크리스마스 분위기을 내었습니다.<br>
&nbsp;&nbsp;&nbsp;&nbsp;크리스마스 이미지 카드 안에는 크리스마스를 기준으로 D-day를 설명하여, 크리스마스까지 얼마나 남았는지 카운트 다운을 합니다.<br>
&nbsp;&nbsp;&nbsp;&nbsp;하단에는 각 이벤트 페이지를 소개하고, 각 이벤트 페이지로 이동할 수 있는 버튼을 제공합니다.<br><br>

5. 편지 보내기 페이지<br>
&nbsp;&nbsp;&nbsp;&nbsp;편지 보내기 기능은 사용자가 형식에 맞게 내용을 입력하고, 메일을 보내는 방식으로 만들었습니다.<br>
&nbsp;&nbsp;&nbsp;&nbsp;'EmailJS' 기능을 이용하여 실제 메일을 보낼 수 있게 만들었으며, 다른 이메일 웹페이지로 이동하지 않아도 페이지 내에서 바로 메일을 보낼 수 있게 만들었습니다.<br>
&nbsp;&nbsp;&nbsp;&nbsp;직접 크리스마스 편지를 보내 보며, 시험해 보시길 바랍니다.<br><br>

7. 트리 꾸미기 페이지<br>
&nbsp;&nbsp;&nbsp;&nbsp;트리 꾸미기 페이지에서는 사용자가 직접 트리 장식을 이용해 크리스마스 트리를 꾸밀 수 있습니다.<br>
&nbsp;&nbsp;&nbsp;&nbsp;트리 장식을 원하는 위치에 드래그 하여 놓을 수 있습니다. 또한 트리 장식에는 메모 기능이 있어, 장식을 트리 위에 놓으면 메모를 입력하는 창이 뜹니다.<br>
&nbsp;&nbsp;&nbsp;&nbsp;사용사는 원하는 메모를 입력하여 저장할 수 있고, 장식된 트리 장식을 다시 클릭하여 메모 내용을 확인할 수 있습니다.<br>
&nbsp;&nbsp;&nbsp;&nbsp;꾸민 트리는 웹페이지를 나갔다 와도 저장이 되며, 더 나아가 다른 사용자와 해당 페이지를 공유할 수 있습니다.<br><br>

9. 미니 게임 페이지<br>
&nbsp;&nbsp;&nbsp;&nbsp;미니 게임 페이지에서는 크리스마스 분위기에 맞는 미니 게임을 즐길 수 있습니다.<br>
&nbsp;&nbsp;&nbsp;&nbsp;미니 게임은 총 3가지가 있으며, 종류는 '하늘에서 선물 받기', '산타 피하기', '눈송이 클릭' 게임이 있습니다.<br>
&nbsp;&nbsp;&nbsp;&nbsp;사용자는 각 게임 중 원하는 게임을 선택하여 플레이할 수 있으며, 각 게임마다 최고 기록을 갱신하면서 게임을 즐길 수 있게 하였습니다.
</p>
